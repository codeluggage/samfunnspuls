# Røde Kors — Website UI kit

A high‑fidelity static recreation of `rodekors.no`‑style marketing surfaces, built with the foundations in `../../colors_and_type.css` and assets from `../../assets/`.

- `index.html` — interactive landing page (donate flow, header nav, hero, value cards, footer)
- `Header.jsx` — fixed top bar with logo, nav, language switcher, donate CTA
- `Hero.jsx` — `HeroSection` recreation (centered title + lead + dual CTAs over the coral wave background)
- `Card.jsx` — content card variants (default, tinted, with accent border)
- `Button.jsx` — primary / secondary / tertiary / danger
- `Footer.jsx` — multi‑column footer with contact info and crisis line
- `Donate.jsx` — donation amount picker + form (Donor‑style component)

**Source:** ported from `norwegianredcross/DesignSystem/src/components/HeroSection`, `Header`, `Footer`, `Donor` and `pages/Home`.
